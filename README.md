# Калькулятор системных ставок

React-приложение для расчёта потенциального выигрыша по системным ставкам (например, 2/3, 3/4 и т.д.).

## Возможности

-   Ввод коэффициентов для отдельных событий
-   Выбор типа системной ставки (например, «2 из 3», «3 из 5» и др.)
-   Указание общей суммы ставки
-   Автоматический расчёт всех возможных комбинаций
-   Отображение выплаты по каждой выигрышной комбинации и итогового возврата
-   Поддержка граничных случаев: ни одна комбинация не выиграла, выиграли все и т.п.

## Технологии

-   React.js (без TypeScript)
-   Redux (для управления состоянием: коэффициенты, тип системы, сумма ставки)
-   Jest / React Testing Library (юнит-тесты)

## Установка и запуск

```bash
npm install
npm start
```

Приложение будет доступно по адресу http://localhost:3000
