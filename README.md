# Динамическая фильтрация каталога товаров

React-приложение с динамической фильтрацией товаров по категории, бренду, цене и рейтингу.

## Возможности

-   Фильтрация товаров в реальном времени
-   Множественные фильтры: категория, бренд, диапазон цен, рейтинг
-   Сортировка по цене, рейтингу и популярности (бонус)
-   Адаптивный интерфейс: удобен как на мобильных, так и на десктопах
-   Индикация загрузки и сообщение при отсутствии результатов

## Технологии

-   React.js (без TypeScript)
-   Redux (для управления состоянием фильтров и сортировки)
-   Jest / React Testing Library (юнит-тесты)

## Установка и запуск

```bash
npm install
npm start
```
